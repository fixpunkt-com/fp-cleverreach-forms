# Cleverreach Forms

Fehler werden in den Log als Warnung geschrieben und im Frontend / beim Absenden einfach ignoriert.

fp_cleverreach muss konfiguriert sein!

Diese Extension bietet einen ext:form Finisher an,
aber die Login-Daten werden sich von fp_cleverreach gezogen.
Dort müssen die Anmeldedaten in der Extension Configuration gesetzt sein!
